## 01-2 컴퓨터 구조의 큰 그림
### 컴퓨터 구조
- 컴퓨터가 이해하는 정보 (데이터, 명령어)
- 컴퓨터의 네 가지 핵심 부품 (CPU, 주기억장치, 보조기억장치, 입출력장치)

<br>**CPU**<br>
> 메모리에 저장된 명령어 읽기, 실행
- **산술논리연산장치**(ALU, Arithmetic Logic Unit) : 계산만 실행
- **레지스터**(register) : 작은 임시 저장장치
- **제어장치**(CU, Control Unit) : *제어신호*(메모리 읽기, 쓰기)를 내보내고, 명령어 해석
<details>
<summary>메모리 읽기</summary>
<div markdown="1">
메모리에 저장된 값을 읽을경우
</div>
</details>

<details>
<summary>메모리 쓰기</summary>
<div markdown="1">
메모리에 값을 저장할 경우

</div>
</details>

<br>**주기억장치(메모리)**<br>
> 현재 실행중인 명령어, 데이터 저장. (전원 꺼지면 내용 사라짐)

<br>**보조기억장치(SSD, 하드디스크 등)**<br>
> 내용 저장. (전원 꺼져도 내용 저장)

<br>**입출력장치(마우스, 키보드 등)**<br>
> 컴퓨터 외부에 연결, 컴퓨터 내부와 정보 교환

### <br>**메인보드(마더보드), 시스템버스**<br>
> 컴퓨터의 핵심 부품(CPU, 메모리, 보조기억장치, 입출력장치)은 모두 *메인보드*에 연결<br>

> 시스템 버스 : 컴퓨터의 핵심 부품들이 서로 정보를 주고 받는 통로
